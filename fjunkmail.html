F Junk Mail url

I built F Junk Mail one friday night. I received my mail late that afternoon and it was nothing but Charter junkmail. Fed up, I put together F Junk Mail to spam Charter back.

I used Stripe for payment processing and Sincerely's API for sending the postcards. It made for a fun friday night. The site trended on Hacker News' frontpage for a significant part of Saturday and it continues to be used by others around the United States to retaliate against junkmail.

Technologies include:

  • Sinatra
  • Angular.js
  • Stripe
  • Sincerely API
  • Redis
