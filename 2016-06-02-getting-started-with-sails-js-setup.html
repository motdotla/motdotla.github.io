Getting Started With SailsJS

In this article, I show you how to setup SailsJS.

Install NodeJS

Visit https://nodejs.org and download the LTS version of Node. At the time of writing this was v.4.4.5 LTS.

Follow the installer to completion.

Once completed open up Terminal and check your node version. 

node -v

Install Postgres

Download Postgres.app onto your machine and follow the installation process.

Install SailsJS

sudo npm -g install sails

Create a project. 

sails new yourapp

Enter the project and run the server. 

cd yourapp
npm start

Visit http://localhost:1337

That's it!

You're all set. You have a good foundation for building out your SailsJS application.
