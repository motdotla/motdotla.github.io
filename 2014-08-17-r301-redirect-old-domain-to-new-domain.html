r301: Redirect Old Domain to New Domain

I needed a way to redirect my old domain to a new domain. I didn't find a solution I really liked so I built my own - r301.

I looked at some options like the following.

  • jekyll page redirects
  • using an htaccess file
  • using a php script.

The options were good in some scenarios, but not as easy as I would like to quickly switch my entire domain over. r301 makes it easy.

r301 let's you quickly 301 permanently redirect from your old domain to your new domain.

It is a small application that you configure and deploy to heroku in just 5 minutes. I hope you find it useful. Instructions for configuration and deployment are in the README.
