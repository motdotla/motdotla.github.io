Node-lambda is a command line tool for locally running and remotely deploying node.js applications to Amazon Lambda.

I built node-lambda soon after the release of Amazon Lambda. We were experimenting with using Lambda at Rebelmail and I spent a few evenings coding together the basics of the tool.

It's sense become a useful alternative to more powerful tools like the excellent Serverless Framework. It proves to be a quick way to get up and running with Amazon Lambda.
