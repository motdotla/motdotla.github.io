Carve API url

Carve API makes it easy to convert PDFs to an array of PNGs. Simply make a web request. 

https://carve-api.herokuapp.com/api/v0/documents/create.json?url=http://mot.la/assets/resume.pdf&webhook=http://requestb.in/10c8e4o1

The url must be a url to a PDF file. The webhook must be a url that can accept a POST request with the processed payload - requestb.in is handy for this.

The application is entirely open source. You can host it yourself in just one, two clicks.

Those two clicks will install all of the following - the full stack for Carve API.

  • carve-api - for letting the developer make conversion requests.
  • carve-worker - for converting the PDF to an array of PNGs in the background.
  • carve - for doing the operating system level conversion of PDF to PNGs.
  • heroku-buildpack-mupdf - for installing the conversion tool mudraw.
  • mupdf-binary - for installing onto Heroku's servers.
  • IronMQ - for data and queue management.
